# /wf-release
Tags: #workflow #release #demo-readiness

## Goal
Verify the system is safe and coherent before demo or internal rollout.

## Checklist
- Story output is consistent
- Evaluation gate is enforced
- Approval gate is clear
- Jira payload mapping is correct
- Slack notifications are understandable
- Traces and failure logs are visible

1. /story-eval
2. /approval-flow
3. /jira-action
4. /slack-notify
5. /trace-observe
